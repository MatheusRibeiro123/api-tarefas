# Aplicativo de Gerência de Tarefas

Projeto em desenvolvimento de uma API para gerenciamento de tarefas, com foco em praticar conceitos de backend, organização de rotas, modelos, banco de dados e operações CRUD.

## Status do projeto
🚧 Em desenvolvimento

Este projeto ainda está em fase de criação e aprendizado, então algumas funcionalidades podem estar incompletas, em teste ou sofrer alterações ao longo do tempo.

## Objetivo
O objetivo deste projeto é estudar e aplicar na prática conceitos como:
- criação de rotas
- organização com Flask
- integração com banco de dados
- relacionamento entre usuários e tarefas
- operações CRUD
- estruturação de API

## Funcionalidades atuais
- cadastro de usuários
- listagem de usuários
- criação de tarefas
- listagem de tarefas
- listagem de uma tarefa específica por usuário

## Funcionalidades planejadas
- atualização de tarefas
- exclusão de tarefas
- autenticação de usuários
- validação de dados
- melhorias na organização do projeto

## Tecnologias utilizadas
- Python
- Flask
- SQLAlchemy
- SQLite

## Observação
Este projeto foi criado com fins de estudo e prática.  
A ideia é evoluí-lo aos poucos conforme o aprendizado for avançando.
